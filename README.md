# Customer Segmentation Dashboard

## Overview
An industry-grade machine learning and analytics dashboard for customer segmentation using:

- Python
- Streamlit
- Pandas
- Scikit-learn
- Plotly

## Features
- 100,000+ customer records
- K-Means clustering
- PCA visualization
- Elbow method optimization
- Customer behavioral segmentation
- Demographic analysis
- Regional insights
- Loyalty analysis
- Executive dashboard UI
- Downloadable reports

## Business Insights
- High-value customer groups
- At-risk customers
- Premium segment opportunities
- Targeted marketing recommendations

## Tech Stack
- Streamlit
- Pandas
- Scikit-learn
- Plotly
- Matplotlib
